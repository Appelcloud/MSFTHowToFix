## Description
This guide shows how to enable Authenticator App OTP in Entra ID portal

### Guide
1.) Go to "[Entra.Microsoft.com](https://entra.microsoft.com)"

2.) Sign in with an account that has that has the right priviliges for Authentication Methods.

3.) Expand **Protection** and Click on **Authentication Methods**

![Pic01](/img/Authenticator-App-Enable-OTP/Pic01.png)

4.) In the **Manage** pane, select **Policies** and click on **Microsoft Authentication**. 

![Pic02](/img/Authenticator-App-Enable-OTP/Pic02.png)

5.) Click on **Configure** tab and select **Yes** in the *Allow use of Microsoft Authenticator OTP*

![Pic03](/img/Authenticator-App-Enable-OTP/Pic03.png)

6.) Click **Save**
