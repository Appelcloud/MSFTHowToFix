## Description
This guide shows how to check if Authentication App is enabled or disabled as a Authentication Methods in Entra ID portal

### Guide
1.) Go to "[Entra.Microsoft.com](https://entra.microsoft.com)"

2.) Sign in with an account that has that has the right priviliges for Authentication Methods.

3.) Expand **Protection** and Click on **Authentication Methods**

![Pic01](/img/Authenticator-app-tenant-status/Pic01.png)

4.) In the **Manage** pane, select **Policies** and click on **Microsoft Authentication**. 

:::tip
You can see whether is it Enabled or not from this Page, by looking at **Enabled** column
:::
![Pic02](/img/Authenticator-app-tenant-status/Pic02.png)
